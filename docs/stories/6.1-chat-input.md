# Story 6.1: Zone de saisie texte (Chat Input)

**Status:** Approved
**Story:**
**As a** User,
**I want** a text input field on the A2UI,
**so that** I can communicate with my crew by typing.

## Acceptance Criteria
1.  Une zone de saisie texte (input) est ajoutée en bas de l'interface `a2ui/index.html`.
2.  Le style CSS respecte le thème "Obsidian" (transparent, bordure accentuée).
3.  Appuyer sur "Entrée" ou cliquer sur le bouton "Envoyer" publie un message `narrative.text` sur le WebSocket.
4.  Le message envoyé a pour émetteur `agent_id: "user"`.
5.  Validation : Le message apparaît dans les logs du H-Core après envoi.

## Tasks / Subtasks
- [ ] Task 1: HTML/CSS Input (AC: 1, 2)
    - [ ] Ajouter `<input id="chat-input">` et `<button id="chat-send">` dans `index.html`.
    - [ ] Styliser les éléments dans `style.css` (Position absolue en bas).
- [ ] Task 2: WebSocket Integration (AC: 3, 4)
    - [ ] Dans `network.js`, implémenter une méthode `sendUserMessage(text)`.
    - [ ] Construire l'enveloppe H-Link conforme v1.1.
- [ ] Task 3: Event Listeners
    - [ ] Lier les événements claviers et clics aux fonctions d'envoi.
