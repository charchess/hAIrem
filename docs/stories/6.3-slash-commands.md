# Story 6.3: Slash Commands (Direct Control)

**Status:** Approved
**Story:**
**As a** Power User,
**I want** to use commands like `/expert-domotique turn_on light.salon`,
**so that** I can trigger actions faster and more reliably for routine tasks.

## Acceptance Criteria
1.  L'interface de chat détecte le préfixe `/`.
2.  Si un message commence par `/`, il est converti en type `expert.command` au lieu de `narrative.text`.
3.  Le premier mot après le `/` est utilisé comme `recipient.target`.
4.  Le reste de la ligne est passé comme `payload.content`.
5.  Validation : Taper `/ping` renvoie une réponse "pong" immédiate de l'agent visé.

## Tasks / Subtasks
- [ ] Task 1: Command Parser (AC: 1, 2)
    - [ ] Implémenter une fonction `parseSlashCommand(text)` dans `network.js`.
- [ ] Task 2: Direct Routing (AC: 3, 4)
    - [ ] Envoyer le message directement au channel de l'expert (`target`).
- [ ] Task 3: Feedback (AC: 5)
    - [ ] Afficher une confirmation de commande dans l'historique.
