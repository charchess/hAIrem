# Story 3.1: Moteur de rendu de calques (A2UI Stage)

**Status:** Draft
**Story:**
**As a** User,
**I want** to see characters composed of superposed image layers,
**so that** expressions and poses can change dynamically without reloading the whole character.

## Acceptance Criteria
1.  Une structure HTML5/CSS3 est créée pour "The Stage" (Le plateau).
2.  Le système supporte au moins 4 calques (Z-layers) : Background (Z-0), Body (Z-10), Face/Expression (Z-20), Dialogue Overlay (Z-30).
3.  Utilisation de CSS Transitions ou Framer Motion pour des changements de poses fluides (opacité/glissement).
4.  Une démo statique montre "La Renarde" changeant d'expression (ex: Neutral -> Happy).

## Tasks / Subtasks
- [ ] Task 1: Stage Structure (AC: 1, 2)
    - [ ] Créer `apps/a2ui/index.html` et `apps/a2ui/style.css`.
    - [ ] Définir le conteneur `#the-stage` avec les calques positionnés en absolu.
- [ ] Task 2: Component logic (AC: 3)
    - [ ] Créer un script `apps/a2ui/js/renderer.js` pour gérer l'affichage.
    - [ ] Implémenter une fonction `renderPose(agent_id, pose_data)`.
- [ ] Task 3: Placeholder Assets
    - [ ] Ajouter des images de test (ou formes géométriques colorées) pour valider la superposition.
