# Story 5.3: Boucle d'action réelle (Command -> HA -> Response)

**Status:** Approved
**Story:**
**As a** User,
**I want** to say "Allume la lumière" and see it happen while hearing "C'est fait !",
**so that** hAIrem est mon interface domotique principale.

## Acceptance Criteria
1.  Un flux complet est validé : User -> Renarde -> Expert-Domotique -> Home Assistant.
2.  L'expert renvoie une `expert.response` sur Redis après l'action HA.
3.  La Renarde capte cette réponse et génère un message narratif de succès ou d'échec.
4.  L'interface A2UI affiche le texte final.

## Tasks / Subtasks
- [ ] Task 1: Inter-agent Coordination
    - [ ] Implémenter la logique où Renarde délègue à l'Expert via `expert.command`.
- [ ] Task 2: Action Execution
    - [ ] Brancher les outils réels sur les appels HA.
- [ ] Task 3: Final Report
    - [ ] Générer la réponse narrative finale.
