# Epic 7: Agent Dashboard & Control

**Goal:** Visualiser et gérer l'équipage hAIrem.

## User Stories

### Story 7.1: Vue liste des agents
**As a** User,
**I want** to see which agents are active,
**So that** I know my crew.

**Acceptance Criteria:**
- [ ] Grid or list view of agents.
- [ ] Status indicator (Idle/Speaking/Thinking).

### Story 7.2: Logs Temps Réel
**As a** Developer,
**I want** to see system logs in the UI,
**So that** I can debug without opening a terminal.

**Acceptance Criteria:**
- [ ] Component displaying system.log messages.
